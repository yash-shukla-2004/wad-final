PICT B.E. Course Structure

This is the course structure for the Bachelor of Engineering (B.E.) at PICT, Pune.

First Year (FE)

  • Engineering Mathematics-I
    Basic calculus, linear algebra, and differential equations.
  • Engineering Physics
    Concepts of mechanics, waves, optics, and modern physics.
  • Engineering Chemistry
    Atomic structure, chemical bonding, and materials science.
  • Basic Electrical Engineering
    Ohmâ€™s Law, circuits, and electrical machines.
  • Programming in C
    Introduction to programming logic and C language.

Second Year (SE)

  • Data Structures
    Arrays, linked lists, stacks, queues, trees, graphs.
  • Computer Organization
    Architecture of CPUs, memory, and I/O systems.
  • Discrete Mathematics
    Logic, sets, relations, and graph theory.
  • Object-Oriented Programming (C++)
    Classes, inheritance, polymorphism.
  • Digital Electronics
    Logic gates, flip-flops, multiplexers.

Third Year (TE)

  • Database Management Systems
    SQL, normalization, transaction management.
  • Operating Systems
    Processes, scheduling, memory management, file systems.
  • Computer Networks
    OSI model, TCP/IP, routing, protocols.
  • Software Engineering
    SDLC, UML, testing methodologies.
  • Web Technologies
    HTML, CSS, JavaScript, jQuery, PHP.

Final Year (BE)

  • Machine Learning
    Supervised and unsupervised learning, algorithms.
  • Cloud Computing
    Virtualization, cloud services, deployment models.
  • Information & Cyber Security
    Cryptography, network security, firewalls.
  • Electives (e.g., AI, Blockchain)
    Specialized topics as per interest and demand.
  • Project Work
    Final year project and seminar presentations.
