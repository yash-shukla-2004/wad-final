PICT B.E. Course Structure

First Year (FE)

  • Engineering Mathematics I
    Foundation in calculus and linear algebra.
  • Engineering Physics
    Concepts in mechanics, optics, and modern physics.
  • Basic Electronics
    Introduction to electronic components and circuits.
  • Programming in C
    Basics of C language and problem solving.

Second Year (SE)

  • Data Structures
    Efficient ways to store and access data.
  • Computer Architecture
    Study of CPU, memory, and I/O systems.
  • Discrete Mathematics
    Logic, sets, relations, and combinatorics.
  • Object Oriented Programming
    Programming with C++ and object-oriented concepts.

Third Year (TE)

  • Operating Systems
    Concepts of process management, memory, and scheduling.
  • Database Management Systems
    Relational databases and SQL.
  • Software Engineering
    Software development life cycle and design principles.
  • Web Technologies
    HTML, CSS, JavaScript, and server-side programming.

Final Year (BE)

  • Artificial Intelligence
    Introduction to AI concepts and applications.
  • Machine Learning
    Supervised and unsupervised learning techniques.
  • Cloud Computing
    Distributed systems and cloud platforms.
  • Project Work
    Capstone project in chosen specialization.
